What are the task sizes?

Microtasks can be specified in the following sizes:

Size Duration Price Ideas
{{ small_desc }} 1 hour {{ currency }}{{ small_price }} Make dinner reservations for 4 next thursday at...
{{ medium_desc }} 2 hours {{ currency }}{{ medium_price }} Call these 10 companies and discover their CIOs' numbers
{{ halfday_desc }} 4 hours {{ currency }}{{ halfday_price }} Buy 20 Best Buy gift cards and have them sent to my employee's homes
{{ fullday_desc }} 8 hours {{ currency }}{{ fullday_price }} Look over all these resumes, and rank them by this criteria...

Pick the size that reflects how long you would expect an educated person to take to produce a suitable result.

If you're unsure, start with a {{ small_desc }} task and see how it goes. Then, as you become accustomed with the process, increase to larger task sizes.
